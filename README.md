# 114_web_midterm_project
 
# 專案名稱
- 健身房會員報名系統

# 組員
-陳相宇

# 專案簡介
- 填寫姓名、電話、Email 與會員方案，送出後立即預覽報名資料。系統具備表單驗證、資料儲存功能，支援響應式設計

# 使用技術
- HTML、CSS、Bootstrap、JavaScript、Constraint Validation API、DOM 操作

# 功能特色
- 1.表單即時驗證 
- 2.即時報名結果預覽
- 3.資料保存
- 4.響應式設計
